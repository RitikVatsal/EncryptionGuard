# DEVELOPMENT

## Development Mode
Add the `EncryptionGuard\extension\dist` folder in browser's developer mode

In /extension,$npm run dev

## Production Mode
Add the `EncryptionGuard\extension\build` folder in browser's developer mode

In /extension, $npm run build
